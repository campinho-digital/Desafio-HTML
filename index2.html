Desafio: Desenvolvimento de Formulário HTML5

Descrição do Projeto

Seu objetivo neste desafio é criar uma página web com um formulário HTML5 que colete informações pessoais e, quando o usuário preencher o formulário e o enviar, exiba os dados do formulário em uma página HTML separada.


Requisitos Funcionais

Formulário: Crie um formulário HTML5 com os seguintes campos:
  • Nome (input tipo texto)
  • Idade (input tipo número)
  • E-mail (input tipo email)
  • Gênero (input tipo radio button ou select)
  • Uma área de texto para comentários ou observações.

Página de Resultado: Após o usuário preencher o formulário e clicar no botão de envio, ele deve ser redirecionado para uma página separada que exibe os dados do formulário.Essa página de resultado deve ser formatada como uma saída HTML e deve incluir os dados preenchidos pelo usuário.


Requisitos Técnicos

Use somente HTML5 para criar a estrutura das páginas, o formulário e a página de resultado.

Não é necessário usar CSS ou JavaScript neste desafio. Concentre-se apenas na estrutura e no conteúdo das páginas.

Resultado Esperado

Ao concluir o desafio, você deve ter duas páginas da web:

Uma página com um formulário funcional que coleta informações pessoais.

Uma página de resultado que exibe os dados do formulário em uma saída HTML formatada.


Critérios de aceitação

Todas as perguntas e respostas precisam ser exibidas na tela.
